# Hypreact CSS

## Selector Targets

- `workspace`
- `group`
- `window`

Invalid selector targets:

- `slot`

## Window Attribute Selectors

- `window[app_id="..."]`
- `window[title="..."]`
- `window[class="..."]`
- `window[instance="..."]`
- `window[role="..."]`
- `window[shell="..."]`
- `window[window_type="..."]`

## Pseudo-Classes

- `:focused`
- `:floating`
- `:fullscreen`
- `:urgent`
- `:closing`
- `:enter-from-left`
- `:enter-from-right`
- `:exit-to-left`
- `:exit-to-right`

## Pseudo-Elements

No pseudo-elements are currently supported.
